# Getting Started

## Required Environments
* JDK (>=11)

## 1. git clone
```
git clone https://github.com/tngus3722/GOODCHOICE_CURSOR.git
cd GOODCHOICE_CURSOR
```

## 2. build jar
need java (>= 11)
```
./gradlew bootJar 
```

## 3. excute jar
```
java -jar build/libs/*.jar
```
